# vue_corollazero

요구사항

1. 디자인 퍼블리싱은 bootstrap을 이용
2. 상태관리는 pinia를 통해 한다.
3. 이동은 router를 통해 한다.
4. vite를 이용하여 빌드 한다.
5. api호출은 axios로하며, 로컬 테스트를 위한 json반환 서버를 이용한다.
6. component를 분리하여 만든다.
7. 확인창 및 알러트창은 vue.js로 구현한다.
